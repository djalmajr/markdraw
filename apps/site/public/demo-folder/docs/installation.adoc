= Installation

Three install paths depending on your platform — see
xref:getting-started.adoc[Getting Started] if you haven't opened
the app yet.

toc::[]

== macOS

=== Apple Silicon

Download `Markdraw-macos-arm64.dmg`, open the disk image, and drag
Markdraw into Applications. The app is Apple-signed and notarized.

=== Intel

Same flow, but use `Markdraw-macos-x64.dmg`. Auto-update keeps
both architectures in sync once installed.

== Linux

=== AppImage

`chmod +x Markdraw-linux-x64.AppImage` and run it. The binary is
self-contained — no system packages required.

=== Debian / Ubuntu

`sudo apt install ./Markdraw-linux-x64.deb`. Adds a desktop
launcher under "Office" / "Development".

== Windows

The MSI is the recommended path — handles uninstall + auto-update
cleanly. SmartScreen may prompt on first run; click *More info →
Run anyway*.

== After installing

Open xref:getting-started.adoc[Getting Started] for the welcome
flow, or jump straight to xref:usage.adoc[Usage] if you've used
similar tools before.
