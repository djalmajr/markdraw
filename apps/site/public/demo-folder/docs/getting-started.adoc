= Getting Started

A quick tour of AsciiMark — the local-first AsciiDoc and Markdown
viewer with diagrams, math, and a workspace-aware navigator.

toc::[]

== Open a workspace

Drop any folder onto the welcome screen, or click the dropzone to
pick one. AsciiMark scans the tree, hides `node_modules` /  `.git`
by default, and lists `.adoc` and `.md` files in the sidebar.

=== Multiple roots

Open a second folder via *File → Open Folder...* or drag-and-drop.
Both trees stay side by side — handy when reading docs spread
across repositories.

=== File preview

Click any file to open it in the active pane as a preview tab.
Preview tabs use italic so it's clear they'll be replaced by your
next click. Double-click (or start editing) to pin.

== Navigate quickly

Open xref:installation.adoc[Installation] for setup options, or
jump to xref:usage.adoc[Usage] for the daily workflows.

=== Keyboard-first

A few of the most-used bindings:

[cols="1,3"]
|===
| `Cmd/Ctrl+P` | Quick Open (fuzzy find any file)
| `Cmd/Ctrl+Shift+P` | Command Palette (every action)
| `Cmd/Ctrl+Shift+O` | Go to Symbol in this file
| `Cmd/Ctrl+Alt+O` | Go to Symbol in workspace
| `Cmd/Ctrl+\` | Split editor
| `Cmd/Ctrl+.` | Toggle Reader Mode
|===

The full table lives in xref:usage.adoc#shortcuts[Usage > Shortcuts]
and inside the app at `Cmd/Ctrl+/`.
