# Demo Workspace

This folder is used by the AsciiMark marketing site to capture
screenshots of the desktop app in action. The content is intentionally
fictional so it works as a clean visual.

## Layout

- [`docs/getting-started.adoc`](docs/getting-started.adoc) — quick start
- [`docs/installation.adoc`](docs/installation.adoc) — install paths
- [`docs/usage.adoc`](docs/usage.adoc) — daily workflows
- [`docs/architecture.adoc`](docs/architecture.adoc) — diagrams + structure
- [`notes/todo.md`](notes/todo.md) — running checklist
- [`notes/changelog.md`](notes/changelog.md) — release log

## Where to start

New to AsciiMark? Open
[`docs/getting-started.adoc`](docs/getting-started.adoc). Already
familiar with the editor and want to dig deeper?
[`docs/architecture.adoc`](docs/architecture.adoc) covers the
rendering pipeline and the keyboard model.
