Los servidores Model Context Protocol (MCP) amplían el asistente con herramientas adicionales — sistemas de seguimiento de incidencias, herramientas de diseño, documentación y más. Gestiónalos en <strong>Ajustes → Servidores MCP</strong>.

## Añadir un servidor

Añade un servidor local por comando (stdio) o uno remoto por URL (HTTP). Las herramientas de cada servidor se listan para que veas qué ofrece antes de activarlo.

## Servidores descubiertos

Los servidores ya configurados para otras herramientas — Claude, Codex, OpenCode — se descubren automáticamente. Los servidores de ámbito de proyecto piden tu aprobación antes de que Markdraw los ejecute.

## Autenticación

Los servidores que requieren inicio de sesión usan OAuth — haz clic en <strong>Autorizar</strong> y Markdraw abre tu navegador para conceder acceso. Se admiten cabeceras personalizadas para servidores que autentican de otra manera.
