---
title: "Modo lectura / Zen"
description: "Todo lo que necesitas para instalar, configurar y usar Markdraw en los flujos de escritorio y extensión."
---

<code>Cmd/Ctrl+.</code> (o <em>Vista → Alternar modo lectura</em> en la Paleta de comandos) colapsa la barra de herramientas, la barra lateral, el TOC y la barra de estado — dejando solo la vista previa renderizada centrada a un ancho de lectura cómodo. Pulsa el mismo atajo para volver.

<code>F11</code> se acepta como fallback en plataformas donde el SO no lo tiene tomado. El Mission Control de macOS engulle <code>F11</code> por defecto — usa el atajo del punto allí.
