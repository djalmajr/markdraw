Markdraw se comunica con los proveedores de IA directamente desde tu máquina — no hay cuenta ni servidor de Markdraw de por medio. Conecta uno o más desde <strong>Ajustes → Asistente de IA</strong>.

## Claves de API

Pega una clave de API de un proveedor como OpenAI o Anthropic. Las claves se guardan en el llavero de tu sistema operativo, nunca en archivos de texto plano.

## Suscripciones por CLI

Si ya usas una CLI de agente de código como Claude Code o Codex, Markdraw puede reutilizar su suscripción — sin necesidad de una clave de API aparte.

## Proveedores compatibles con OpenAI

Apunta Markdraw a cualquier endpoint compatible con OpenAI — runtimes locales como Ollama y LM Studio, o gateways alojados — indicando un nombre, una URL base y los modelos.

## Modelos, razonamiento y streaming

Elige qué modelos aparecen en el selector, define cuánto esfuerzo de razonamiento usar y activa o desactiva las respuestas en streaming. El selector de modelo del compositor agrupa todo por proveedor.
