## AsciiDoc

Usa Asciidoctor con soporte para admonitions, tablas, bloques de código fuente, xrefs e includes recursivos.

## Markdown

Usa markdown-it con plugins para listas de tareas, notas al pie, listas de definición, emojis, alertas, tablas avanzadas y más.
