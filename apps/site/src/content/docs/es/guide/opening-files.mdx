## Arrastrar y soltar

Suelta una carpeta para poblar el árbol, o suelta un archivo para abrirlo directamente.

## Respetar .gitignore

Activa <strong>Respetar .gitignore</strong> en el dropdown de visibilidad del file-tree (ícono de sliders) para ocultar toda ruta que coincida con el <code>.gitignore</code> del workspace — incluyendo <code>.gitignore</code>s anidados en subdirectorios. Viene <strong>activado</strong> por defecto, así que los artefactos de build (<code>node_modules</code>, <code>dist</code>, <code>target</code>, etc.) quedan fuera del árbol sin configuración. Desactívalo para revelar lo que el bloque de carpetas siempre ignoradas sigue ocultando.

## Modo URL

Las URLs terminadas en <code>.adoc</code> y <code>.md</code> se pueden renderizar directamente.

## Acceso a URL de archivo (extensión)

- Abre <code>chrome://extensions</code>
- Encuentra Markdraw y abre Detalles
- Activa Permitir acceso a URLs de archivo
