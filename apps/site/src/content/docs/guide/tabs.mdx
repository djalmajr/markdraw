---
title: "Tabs — preview vs pinned"
description: "Everything you need to install, configure, and use Markdraw across desktop and extension workflows."
---

Single-click in the file tree opens the file in the <strong>preview slot</strong> — the tab title shows in italic. Click another file and the preview is replaced. Double-click, drag, or just start editing to <strong>pin</strong> the tab; pinned tabs stay until you close them.

The invariant: at most one preview tab per pane. Middle-click and "Open in New Tab" always create a pinned tab.
