Markdraw opens images and PDFs in a builtin viewer — no external app, no leaving the workspace. Enable “Show all files” in the file-tree visibility dropdown to see them in the tree.

## Images

Renders <code>.png</code>, <code>.jpg</code>, <code>.jpeg</code>, <code>.gif</code>, <code>.webp</code>, <code>.avif</code>, <code>.bmp</code>, <code>.svg</code> and <code>.ico</code> with zoom in/out, fit-to-window, and a checkerboard backdrop for transparency.

## PDF

PDFs render every page in a single continuous scroll (no page-flipping), with zoom and fit-to-width. Rendering is bundled, so it looks identical on macOS, Windows, and Linux.

## Edit, Preview, and Split by file type

The Edit / Edit & Preview / Preview toggle adapts to what each file supports:

- <strong>Markdown & AsciiDoc</strong> — edit, split, and preview are all available.
- <strong>Images & PDF</strong> — preview only; editing is disabled.
- <strong>Plain text</strong> (txt, json, yaml, …) — edit only; there is no rendered preview.
- <strong>Unsupported files</strong> (binaries that can't be rendered or read as text) show an “unsupported format” notice.

## Zoom and pan

Images and PDFs open in a viewer with zoom and drag-to-pan — the same controls as the diagram viewer. Use the toolbar, the scroll wheel, or a trackpad pinch.

## Inline images in documents

Click an image inside a Markdown or AsciiDoc preview to set its size — small, fit, full width, or original — or to open it full-screen and zoom in.
