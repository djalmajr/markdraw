## Close behavior (desktop)

By default, clicking the window close button (X / Cmd+W) hides Markdraw to the system tray — the process keeps running so reopening is instant. Open the toolbar menu and pick <strong>Window → Quit app</strong> to make the close gesture actually terminate the process. <code>Cmd+Q</code> always quits regardless of this setting.
