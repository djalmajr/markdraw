title: "Reader / Zen mode"
description: "Everything you need to install, configure, and use Markdraw across desktop and extension workflows."
---

<code>Cmd/Ctrl+.</code> (or <em>View → Toggle Reader Mode</em> in the Command Palette) collapses the toolbar, sidebar, TOC, and status bar — leaving only the rendered preview centered at a comfortable reading width. Press the same shortcut to come back.

<code>F11</code> is accepted as a fallback on platforms where it isn't already taken by the OS. macOS Mission Control swallows <code>F11</code> by default — use the period chord there.
