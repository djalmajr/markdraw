title: "Workspace Symbol Search"
description: "Everything you need to install, configure, and use Markdraw across desktop and extension workflows."
---

<code>Cmd/Ctrl+Alt+O</code> opens a fuzzy-match palette over every heading in every doc of the workspace — not just the active file. Type the heading text or the file path to scope; pick a row and the file opens, scrolled to the right line, with the TOC active highlight already on the matching item.
