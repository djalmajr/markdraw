## Drag and Drop

Drop a folder to populate the tree, or drop a file to open it directly.

## Respect .gitignore

Toggle <strong>Respect .gitignore</strong> in the file-tree visibility dropdown (slider icon) to hide every path matched by the workspace's <code>.gitignore</code> — including nested <code>.gitignore</code> files inside subdirectories. The toggle ships <strong>on</strong> by default so build artifacts (<code>node_modules</code>, <code>dist</code>, <code>target</code>, etc.) stay out of the tree without configuration. Flip it off to reveal everything the always-ignored folders block still hides.

## URL Mode

URLs ending with <code>.adoc</code> and <code>.md</code> can be rendered directly.

## File URL Access (Extension)

- Open <code>chrome://extensions</code>
- Find Markdraw and open Details
- Enable Allow access to file URLs
