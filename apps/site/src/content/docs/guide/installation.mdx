## Chrome Extension

The Chrome Extension is available on the Chrome Web Store: <a href="https://chromewebstore.google.com/detail/asciimark/dmcihjkjbeckainfkaddpkeghlllmkbk" target="_blank" rel="noreferrer">install Markdraw</a> to render <code>.adoc</code> and <code>.md</code> files in-browser with formatted preview.

## Desktop App

Download the installer for your platform from the Home page downloads section.

## macOS

The macOS app is Apple-signed and notarized. Open the downloaded DMG and drag Markdraw to Applications.

## Windows: SmartScreen

If SmartScreen appears, click <strong>More info</strong> and then <strong>Run anyway</strong>.

## Linux: AppImage

Make the downloaded AppImage executable before running:

`chmod +x Markdraw_*.AppImage`

The <code>.deb</code> package can be installed with your distro package manager.
