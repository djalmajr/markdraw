title: "AI Providers"
description: "Everything you need to install, configure, and use Markdraw across desktop and extension workflows."
---

Markdraw talks to AI providers directly from your machine — there is no Markdraw account or server in between. Connect one or more from <strong>Settings → AI assistant</strong>.

## API keys

Paste an API key for a provider like OpenAI or Anthropic. Keys are stored in your operating system's keychain, never in plain files.

## CLI subscriptions

If you already use a coding-agent CLI such as Claude Code or Codex, Markdraw can reuse its subscription — no separate API key required.

## OpenAI-compatible providers

Point Markdraw at any OpenAI-compatible endpoint — local runtimes like Ollama and LM Studio, or hosted gateways — by giving it a name, base URL, and models.

## Models, reasoning, and streaming

Choose which models appear in the picker, set how much reasoning effort to use, and toggle streamed responses. The model picker in the composer groups everything by provider.
