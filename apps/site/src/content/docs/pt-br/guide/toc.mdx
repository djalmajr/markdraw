title: "Sumário e referências"
description: "Tudo que você precisa para instalar, configurar e usar o Markdraw nos fluxos desktop e extensão."
---

A coluna direita é dividida em duas abas:

- <strong>Sumário</strong> — o índice do documento renderizado. A posição de scroll guia o destaque do título ativo.
- <strong>Referências</strong> — todos os outros documentos do workspace que apontam para o arquivo atual via link Markdown, xref do AsciiDoc, ou include. O contador no badge mostra de cara se o documento ativo é referenciado.

As duas abas ficam montadas independente de qual está visível — a troca é instantânea.
