Servidores Model Context Protocol (MCP) estendem o assistente com ferramentas extras — rastreadores de tarefas, ferramentas de design, documentação e mais. Gerencie-os em <strong>Configurações → Servidores MCP</strong>.

## Adicionando um servidor

Adicione um servidor local por comando (stdio) ou um remoto por URL (HTTP). As ferramentas de cada servidor são listadas para você ver o que ele oferece antes de ativá-lo.

## Servidores descobertos

Servidores já configurados para outras ferramentas — Claude, Codex, OpenCode — são descobertos automaticamente. Servidores com escopo de projeto pedem sua aprovação antes que o Markdraw os execute.

## Autenticação

Servidores que exigem login usam OAuth — clique em <strong>Autorizar</strong> e o Markdraw abre o navegador para conceder acesso. Cabeçalhos personalizados são suportados para servidores que autenticam de outra forma.
