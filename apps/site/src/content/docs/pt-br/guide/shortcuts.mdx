---
title: "Atalhos de teclado"
description: "Tudo que você precisa para instalar, configurar e usar o Markdraw nos fluxos desktop e extensão."
---

No macOS use <code>Cmd</code>; no Linux e Windows use <code>Ctrl</code>. Pressione <code>Cmd/Ctrl+/</code> em qualquer lugar do app para abrir a referência de atalhos ao vivo.

<table className="markdraw-table">
  <thead>
    <tr><th>Atalho</th><th>Ação</th></tr>
  </thead>
  <tbody>
    <tr><td><code>Cmd/Ctrl+P</code></td><td>Abrir rápido — fuzzy-find de arquivo em todos os workspaces abertos</td></tr>
    <tr><td><code>Cmd/Ctrl+Shift+P</code></td><td>Paleta de comandos — todas as ações num só lugar</td></tr>
    <tr><td><code>Cmd/Ctrl+Shift+O</code></td><td>Ir para título — pula para qualquer ponto do documento atual</td></tr>
    <tr><td><code>Cmd/Ctrl+Alt+O</code></td><td>Ir para símbolo no workspace — fuzzy-match de títulos em todos os documentos</td></tr>
    <tr><td><code>Cmd/Ctrl+Shift+F</code></td><td>Buscar em arquivos — procurar conteúdo no workspace</td></tr>
    <tr><td><code>Cmd/Ctrl+.</code></td><td>Alternar modo leitura / Zen (sem chrome, preview centralizado)</td></tr>
    <tr><td><code>Cmd/Ctrl+/</code></td><td>Abrir o modal de atalhos de teclado</td></tr>
    <tr><td><code>Cmd/Ctrl+&#92;</code></td><td>Alternar o segundo painel (dividir / recolher)</td></tr>
    <tr><td><code>Cmd/Ctrl+1 / Cmd/Ctrl+2</code></td><td>Focar o primeiro / segundo painel</td></tr>
    <tr><td><code>Cmd/Ctrl+T</code></td><td>Nova aba no painel ativo</td></tr>
    <tr><td><code>Cmd/Ctrl+W</code></td><td>Fechar a aba ativa</td></tr>
    <tr><td><code>Cmd/Ctrl+Shift+T</code></td><td>Reabrir a última aba fechada</td></tr>
    <tr><td><code>Ctrl+Tab / Ctrl+Shift+Tab</code></td><td>Alternar abas para frente / trás</td></tr>
    <tr><td><code>Cmd/Ctrl+S</code></td><td>Salvar o documento ativo (quando as permissões permitem)</td></tr>
    <tr><td><code>Cmd/Ctrl+F</code></td><td>Buscar dentro do preview / editor atual</td></tr>
  </tbody>
</table>
