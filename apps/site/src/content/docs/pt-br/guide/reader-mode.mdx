---
title: "Modo leitura / Zen"
description: "Tudo que você precisa para instalar, configurar e usar o Markdraw nos fluxos desktop e extensão."
---

<code>Cmd/Ctrl+.</code> (ou <em>Visualização → Alternar modo leitura</em> na Paleta de comandos) recolhe a toolbar, sidebar, TOC e barra de status — deixando só o preview renderizado centralizado numa largura confortável de leitura. Use o mesmo atalho para voltar.

<code>F11</code> é aceito como fallback em plataformas onde ele não está tomado pelo OS. O Mission Control do macOS engole <code>F11</code> por padrão — use o atalho do ponto lá.
