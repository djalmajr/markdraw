O Markdraw fala com provedores de IA diretamente da sua máquina — não há conta nem servidor do Markdraw no meio. Conecte um ou mais em <strong>Configurações → Assistente de IA</strong>.

## Chaves de API

Cole uma chave de API de um provedor como OpenAI ou Anthropic. As chaves são armazenadas no keychain do seu sistema operacional, nunca em arquivos de texto.

## Assinaturas via CLI

Se você já usa uma CLI de agente de código como Claude Code ou Codex, o Markdraw pode reaproveitar a assinatura dela — sem precisar de uma chave de API separada.

## Provedores compatíveis com OpenAI

Aponte o Markdraw para qualquer endpoint compatível com OpenAI — runtimes locais como Ollama e LM Studio, ou gateways hospedados — informando um nome, uma URL base e os modelos.

## Modelos, raciocínio e streaming

Escolha quais modelos aparecem no seletor, defina quanto esforço de raciocínio usar e ligue ou desligue respostas em streaming. O seletor de modelo no composer agrupa tudo por provedor.
